## 🎮 项目简介

《末日防线：代号曙光》是一款 Roguelike 元素的俯视角射击塔防游戏。玩家操控悬浮炮台，抵御一波波机械感染体的进攻，通过局内肉鸽升级和局外 RPG 养成，体验爽快的战斗和深度的策略。

### 核心特色

- **科幻机甲风格**：赛博朋克视觉重构，霓虹色调 + 金属灰
- **悬浮炮台设定**：取消人物模型，专注炮台操作
- **机械感染体敌人**：被病毒侵蚀的机器人，非传统僵尸
- **元素反应系统**：冰 + 火=融化、火 + 雷=超载（微创新）
- **竖屏移动端**：1080x1920 分辨率，适配手机操作
